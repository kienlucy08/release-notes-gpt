**Bug: Fix "Create New Site" Dialog**  
Improve UX/UI | Sites Page  
When creating a new empty site, users are unable to input '.' or '-' characters, causing issues in the UI. The latitude and longitude inputs should be labeled clearly in the dialog box.  
*Details:*  
- '.' character is blocked in the latitude and longitude fields.
- Entering '-' as the first character in latitude or longitude fields triggers an error.
- Latitude and longitude fields lack clear labels in the dialog box.  
*Impact to Users:*  
- Users can now input '.' character in latitude and longitude fields.
- Leading '-' input in latitude or longitude fields will not cause errors.
- Clear labeling of latitude and longitude fields for better input understanding.
