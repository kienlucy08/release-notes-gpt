Improve UX/UI | QA Dashboard  
Column sorting configurations on the QA Dashboard tabs will now be saved per user and per organization, persisting even after navigating away from the dashboard.  
*Details:*  
• Users' column sort configurations on QA Dashboard tabs will be saved and restored upon revisiting the dashboard.  
• Sort preferences will be unique to each user account and organization.  
• Implementation will include storing tab ID, column ID, sort direction, and multi-column sort order if applicable.  
• Data will be stored based on user ID and organization ID, potentially in the database for cross-device and cross-browser consistency.  
*Impact to Users:*  
• Users will no longer lose their column sort configurations when navigating away from the QA Dashboard.  
• Each user in the same organization will have their own saved sort preferences.  
• Users with multiple organization memberships will have independent sort configurations per organization.  
• Improved user experience with personalized and consistent column sorting across all QA Dashboard tabs.
