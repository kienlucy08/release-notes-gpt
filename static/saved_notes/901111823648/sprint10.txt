Platform Expansion  
Introduce a queue/job system for image processing and machine learning tasks, particularly for passing images to Roboflow for detection using API keys.  
*Details:*  
- Establish a file queue for image processing operations
- Create a job to send images to Roboflow for detection, starting with a hardcoded model

- Process involves fetching image data, passing it to Roboflow, waiting for inference results, and storing data in a detection table

*Impact to Users:*  
- Enables efficient image processing and machine learning operations
- Simplifies image detection and classification tasks
- Provides a structured approach for managing models and inference data
